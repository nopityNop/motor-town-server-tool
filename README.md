# Motor Town Server Tool

## Description

Simple tool for Administrators of Motor Town game servers. Includes all functionality provided by the Web API

## Configuration

Configuration is stored in:
- Windows: `%APPDATA%\com.motor-town-server-tool.app\api_config.json`

*This program has only been tested on Windows*

## Usage Instructions

If you can't figure this out...thats crazy.

## Building the Application

To build the application:

```bash
# Install dependencies
deno install

# Run in development mode
deno run tauri dev

# Build for production
deno run tauri build
```
